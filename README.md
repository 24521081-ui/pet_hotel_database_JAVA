# Hệ thống quản lý Pet Hotel - Java Desktop App

Đây là dự án ứng dụng desktop Java cho hệ thống quản lý Pet Hotel, do nhóm Bún Bò thực hiện.

Hiện tại, dự án đã xây dựng được cơ sở dữ liệu cơ bản, phục vụ cho các chức năng quản lý chính của hệ thống như khách hàng, thú cưng, nhân viên, chi nhánh, đặt phòng, dịch vụ, hóa đơn và thanh toán.

Cơ sở dữ liệu là nền tảng để tiếp tục phát triển các chức năng của ứng dụng desktop trong các giai đoạn tiếp theo.

## Công nghệ sử dụng (dự kiến)
- Java
- Oracle Database
- JDBC
- Maven

## Tình trạng hiện tại
- Đã hoàn thành thiết kế cơ sở dữ liệu cơ bản
- Đang tiếp tục phát triển các chức năng của ứng dụng desktop
